一、哈希表

1、哈希表构造散列表并求平均查找长度

问题

将关键字序列（7、8、30、11、18、9、14）散列存储到散列表中。散列表的存储空间是一个下标从0开始的一维数组，散列函数为： H(key) = (key * 3) MOD 7，处理冲突采用线性探测再散列法，要求装填（载）因子为0.7。

（1）请画出所构造的散列表。

（2）分别计算等概率情况下查找成功和查找不成功的平均查找长度。

解答

装载因子是指所有关键子填充哈希表后饱和的程度，它等于 填入表中的关键字总数/哈希表的长度。根据题意，我们可以确定哈希表的长度为 L = 7/0.7 = 10；因此此题需要构建的哈希表是下标为0~9的一维数组。查找不成功的次数就直接找关键字到下一个地址关键字为空的距离，但根据散列函数地址为MOD7，因此初始只可能在0~6的位置，

一、数据传输率

1、给定带宽求数据传输率

问题

​ 一个带宽为4 KHz 的理想低通通道，其允许的最高码元传输速率为多少？

​ 采用8个相位，每相位取2种幅度值的调制技术，可以获得的数据传输率是多少？

解答

  • 数据传输速率：在数值上等于每秒传输构成数据代码的二进制比特数,单位为比特/秒(bit/second),记作bps.

  • 信号传输速率：也称码元率、调制速率或波特率,表示单位时间内通过信道传输的码元个数,单位记做Baud.

  • 信道容量：信道在单位时间内可以传输的最大信号量,表示信道的传输能力.信道容量有时也表示为单位时间内可传输的二进制位的位数（称信道的数据传输速率,位速率）,单位为位/秒（b/s）,简记为bps。

因为公式 若信道带宽为W,则奈氏定理的最大码元速率为： B = 2W （Baud）

所以带宽为4K的理想低通信道最大码元速率为：2 × 4000 = 8000 （Baud）

因为采用8个相位，每相位取2种幅度值的调制技术

说明每个信号有2×8 =16种变化，每个码元携带log(2, 16) = 4比特的信息

所以最高数据传送速率 Rmax = Bmax × n = 8000 × 4 = 32000(b/s)

三、IP地址分类的意义

1、IP地址分类的意义

为了方便对 IP 地址的管理，将 IP 地址基本分为三大类，每类地址的分类与含义如下：

  • A 类：前 8 位表示网络 ID，后 24 位表示主机 ID；该地址分配给政府机关单位使用。
  • B 类：前 16 位表示网络 ID，后 16 位表示主机 ID；该地址分配给中等规模的企业使用。
  • C 类：前 24 位表示网络 ID，后 8 位表示主机 ID；该地址分配给任何需要的人使用。

除了上述的 A、B、C 三类地址以外，还有两类隐藏地址，即 D 类地址和E类地址：

  • D 类：不分网络 ID 和主机 ID；该地址用于多播。
  • E 类：不分网络 ID 和主机 ID；该地址用于实验。

一、指针

先说一个特殊的场景，就是在声明一个指针时候，是要加上星号的，比如：

int *a = NULL;

这里的星号没别的意思，就是告诉编译器，a是个指针。而“&”是一个取地址的运算符

int a = 1;
printf("变量a自己的地址是:  %x\n", &a);

输出的结果是 22feb4 表示变量a存放的地址

用这个五个方面来说明指针：己址、己值、他值、他址、他型

#include<bits/stdc++.h>
using namespace std;
int main(void)
{
	int *a = NULL;
	printf("指针变量a的地址是:  %x\n", &a);
	int b = 1;
	printf("指针变量b的地址是:  %x\n", &b);
	printf("---------------------------------\n");
	a = &b;
	printf("指针变量a自己的地址是:  %x\n", &a);
	printf("指针变量a自己的值是:  %x\n",a);
	printf("指针变量*a的值是:  %x\n",*a);
	return 0; 
}

运行结果

  • 己址：表示自己真实的内存地址

  • 己值：表示自己真实内存地址存放的值

  • 他值：表示自己加*后可以取到自己真实内存地址存放的值

  • 他址：表示自己真实内存地址存放的值就是他址

  • 他型：表示声明指针时 int *a = NULL; int是指针加星后要取的类型

*和->的作用一样 都是用来获取他值

符号.和->的作用和区别

  1. A.B则A为对象或者结构体； 点号（.）：左边必须为实体。
  2. A->B则A为指针，->是成员提取，A->B是提取A中的成员B， 箭头（->）：左边必须为指针；

#include<bits/stdc++.h>
using namespace std;
typedef struct Node{
	int data;
	struct Node *next;
}LinkQueueNode,*LinkQueue;

int InitLinkQueue(LinkQueue &LQ){
	LQ = (LinkQueue)malloc(sizeof(LinkQueueNode));
	if(LQ == NULL) return 0;
	LQ->next = LQ;
	return 1;
} 

int EnterQueue(LinkQueue &LQ,int x){
	LinkQueue temp = (LinkQueueNode*)malloc(sizeof(LinkQueueNode));
	temp->data = x;
	temp->next = LQ->next;
	LQ->next = temp;
	LQ = temp;
	return 1;
}

int DeleteQueue(LinkQueue &LQ,int &x){
	if(LQ->next == LQ) return 0;
	LinkQueue temp = LQ->next->next;
//	if(LQ == temp) LQ = temp->next;
	LQ->next->next = temp->next;
	x = temp->data;
	free(temp);
	return 1; 
}

void forQueue(LinkQueue LQ)
{
	LinkQueue head = LQ;
	LinkQueue q = head->next; 
	while(head != q){
		cout<< q->data <<endl;
		q = q->next;
	} 
	cout<<head->data<<endl; 
}

int main()
{
	LinkQueue LQ;
	InitLinkQueue(LQ);
	EnterQueue(LQ,100);
	EnterQueue(LQ,200);
	EnterQueue(LQ,300);
	forQueue(LQ);
	cout<<"-----------------"<<endl; 
	int x; 
	DeleteQueue(LQ,x);
	DeleteQueue(LQ,x);
	DeleteQueue(LQ,x);
	forQueue(LQ);
	return 0;	
}

三、判断数字序列的出栈顺序是否合法

1、思路如下

左边为模拟栈, 右边为题目给定数字序列

(1) 模拟栈给定序列的栈顶数字不等于题目所给的数字序列的开头时，数字进入。

(2) 模拟栈中的栈顶数字等于题目所给的数字序列的开头时，模拟栈和数字序列的首数字同时出栈;

之后，再进行下一个数字的比较，反复1、2过程。

(3) 当题目所给的数字序列全部入栈后，判断模拟栈中的元素是否为空

  • 若为空，说明数字序列合法
  • 若不为空，说明数字序列不合法

给定输出顺序用栈存可以,用队列存储也可以，下列代码是用栈存储，C++实现。

#include<bits/stdc++.h>
using namespace std;
int main(){	
	stack<int> inputStack,s;
	int a;
	while(cin>>a){
		if(a == -1) break;
		inputStack.push(a);
	} 
	int n = inputStack.size();
	for (int i = 1; i <= n; i++){
		s.push(i);
		while(s.size() !=0 && inputStack.top() == s.top()){
			s.pop();
			inputStack.pop();
		}
	}
	if (s.size() != 0 ) cout<<"不合法"<<endl; 
	else cout<<"合法"<<endl; 
	return 0;
}

C语言实现，用数组模拟栈，相关题目 出栈序列的合法性

#include<stdio.h>
int main(){
	int m,n,k;
    scanf("%d %d %d", &m, &n, &k);
    int stack[1010];       //栈 
	int b[1010];           //存放出栈序列 
    while(k--){
    	int flag = 1, index1 = 1, index2 = 1, top = 0; 
    	for(int i = 1; i <= n; i++) scanf("%d",&b[i]);
    	while(true){
            //判断给出的出栈序列如果和1到n增长顺序相同，则出栈
    		if(index1 == b[index2]){
    			index1++;
    			index2++;
			}
            //判断如果栈中有元素，然后栈顶元素也与此时的出栈序列元素相同，则出栈
			else if(top != 0 && stack[top - 1] == b[index2]){
				top--;
				index2++; 
			}
			else{
				if(index1 > n) break;  //已经判断结束，就跳出循环 
				stack[top] = index1;  //元素入栈 
				top++;
				index1++;
				if(top >= m){
					flag = 0;
					break;
				}
			}
		}
		if(flag == 0 || top != 0) printf("NO\n");
		else printf("YES\n");
	}
	return 0;
}
